#
# 权限分为三种
* # 菜单权限-哪些人能看哪些界面 (模块，菜单，按钮)
* # 数据权限-哪些人能看哪些人的数据 (部门上下级，职位上下级，用户上下级)
* # 字段权限-哪些人能看哪些敏感的信息 (岗位级别，敏感字段)
#
# 角色才有权限
#
# 部门，职位，用户只可以分配角色
